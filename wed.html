ALL ABOUT FULL STACK COURSE


A full stack web developer is a person who can develop both client and server software. In addition to mastering HTML and CSS, he/she also knows how to: Program a browser like using JavaScript, jQuery, Angular, or Vue and Program a server like using PHP, ASP, Python, or Node


FRONTEND TECHNOLOGIES

  1. HTML (Hyper text markup language)
  2. CSS (cascading style sheets)
  3. JavaScript

BACKEND TECHNOLOGIES

  1. Java
  2. PHP
  3. Phyton
